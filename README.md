CSS GRID AND FLEXBOX LANDING PAGE

NAME: YUNUSA BAH
MAT#: 22423255

Topic:
Fitness Brand Landing Page (FitLife)

Description:
This project is a responsive landing page built using HTML and CSS.
It demonstrates the use of CSS Grid and Flexbox for modern web layouts.

Where CSS Grid was used:
- Main page layout (container)
- Services section (grid cards)
- Testimonials section

Where Flexbox was used:
- Navigation bar
- Hero section alignment
- Footer alignment

Features:
- Responsive design using media queries
- Grid layout using auto-fit and minmax
- Flexbox for alignment
- Clean and structured layout
